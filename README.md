# 🛍️ Trendy Vogue - Full Stack E-commerce Website

Trendy Vogue is a full-stack fashion e-commerce web application built using React.js, Node.js, Express.js, and MongoDB. It includes a user website, admin panel, and backend API.

---

## 📁 Project Structure

TRENDY-VOGUE/
├── admin/       # Admin Panel (React + Vite)
├── backend/     # Backend Server (Node.js + Express)
└── frontend/    # User Website (React + Vite)

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

git clone https://github.com/khushikayath70-creator/Trendy-Vogue.git
cd Trendy-Vogue

---

### 2️⃣ Frontend Setup

cd frontend
npm install
npm run dev

---

### 3️⃣ Admin Panel Setup

cd admin
npm install
npm run dev

---

### 4️⃣ Backend Setup

cd backend
npm install
node server.js

OR

npm start

---

## 🔐 Environment Variables

Create a `.env` file inside `backend` folder:

MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=4000
